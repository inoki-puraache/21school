# SQL: множества и JOIN-операции

Эта папка посвящена работе с множествами и соединениями таблиц в PostgreSQL.

## Что здесь показано

- `UNION`, `INTERSECT`, `EXCEPT`
- работа с дубликатами в результирующих наборах
- `CROSS JOIN` и `NATURAL JOIN`
- сравнение подзапросов и `JOIN`-подхода
- понимание семантики результата, а не только синтаксиса

## Структура

- `src/ex00` ... `src/ex10`
- в каждой подпапке отдельный `.sql`-файл

## Контекст данных

Используется та же практическая схема с пиццериями, пользователями, меню, визитами и заказами. Скрипт создания базы лежит в [materials/model.sql](./materials/model.sql).

## Почему это полезно в портфолио

Такие примеры хорошо показывают реляционное мышление: умение рассуждать про пересечения, разности, дубликаты и корректное объединение данных.
