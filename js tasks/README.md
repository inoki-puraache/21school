# Механика игры «Морской бой»

В этой папке собрана поэтапная реализация базовой игровой логики на чистом JavaScript.

## Что здесь есть

- модель корабля с учетом попаданий и проверкой потопления
- игровое поле с размещением кораблей и обработкой атак
- игроки разных типов: человек и простой AI
- управляющий класс приложения с пошаговым игровым циклом
- несколько итераций одной и той же идеи по папкам `src/exNN`

## На что стоит смотреть

- объектно-ориентированное проектирование
- работу с состоянием и игровыми правилами
- декомпозицию логики на сущности `Ship`, `Board`, `Player`, `App`
- простую асинхронность и имитацию ходов

## Структура

- `src/ex00` ... `src/ex10` — последовательные версии реализации
- в каждой папке лежат `index.html` и соответствующий `.js`-файл

## Комментарий

Проект сделан как практическая серия небольших итераций, поэтому внутри удобно смотреть не только на итог, но и на эволюцию решения.
