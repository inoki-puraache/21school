# Базовые SQL-запросы

В этой папке собраны компактные SQL-решения поверх небольшой реляционной модели с пользователями, пиццериями, меню, визитами и заказами.

## Что показывает проект

- фильтрацию по датам и условиям
- сортировку по нескольким полям
- использование вложенных подзапросов
- формирование вычисляемых полей
- умение писать понятные запросы даже при ограничениях по синтаксису

## Модель данных

Схема базы лежит в [materials/model.sql](./materials/model.sql). Основные таблицы:

- `person`
- `pizzeria`
- `menu`
- `person_visits`
- `person_order`

## Структура

- `src/ex00` ... `src/ex09`
- в каждой папке один самостоятельный `.sql`-файл

## Зачем это в портфолио

Небольшие запросы тоже хорошо показывают инженерное мышление: аккуратность, понимание реляционной модели и умение получать нужный результат без лишней сложности.
