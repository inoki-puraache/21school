# Продвинутые JOIN-запросы и CTE

В этой папке собраны SQL-примеры, где уже появляются более сложные объединения таблиц и более прикладные сценарии формирования результата.

## Основные темы

- `INNER`, `LEFT`, `RIGHT`, `FULL JOIN`
- работа с `NULL` в отчетных выборках
- поиск пропущенных дат
- переписывание запросов через `CTE`
- объединение справочных и транзакционных данных

## Структура

- `src/ex00` ... `src/ex10`
- в каждой подпапке один `.sql`-файл

## Источник схемы

Скрипт создания базы находится в [materials/model.sql](./materials/model.sql).

## Почему это полезно

Именно такой тип SQL-задач ближе всего к реальной backend- и analytics-практике: нужно собрать данные из нескольких таблиц, аккуратно обработать крайние случаи и отдать понятный результат.
